# Sage-Stubbs-os331-Assessment-2
# Individual classification project
Individual project comparing classical statistical methods and modern machine learning / deep learning models for detecting fraudulent insurance claims.
# Research question
Can we accurately identify fraudulent insurance claims using classic statistical methods and modern machine learning techniques?
# Dataset
Insurance Fraud Detection dataset (Kaggle).
# Methods
- Logistic Regression (GLM)
- Random Forest
- Gradient Boosting
- Neural Network
# Repository Structure
- `scripts/` : R scripts (preprocessing + modelling)
- `results/` : metrics outputs (e.g., `metrics_summary.csv`)
- `report/`  : final report
- `slides/`  : final slide deck
- `data/`    : dataset location
